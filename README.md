# Insightprofit Hub

InsightProfit Project Hub — unified workspace eliminating tab-switching

## Tech Stack

- HTML/CSS/JavaScript

## Getting Started

```bash
git clone https://github.com/rtmendes/insightprofit-hub.git
cd insightprofit-hub
```

Open `index.html` in your browser to view the project.

## Deployment

This project is configured for deployment on [Vercel](https://vercel.com).
